# 운영체제의 분류
## 동시 작업 가능 여부
### 단일 작업(Single Tasking)
- 한 번에 하나의 작업만 처리한다.
- ex) MS-DOS

### 다중 작업(Multi Tasking)
- 여러 작업을 번갈아 수행하여 동시에 처리하는 것처럼 보이게 한다.
- ex) UNIX, Windows

## 사용자의 수
### 단일 사용자(Single User)
- 한 번에 한 명의 사용자만 시스템을 사용할 수 있다.
- ex) MS-DOS

### 다중 사용자(Multi User)
- 여러 사용자가 동시에 시스템에 접근할 수 있다.
- ex) UNIX, Windows Server

## 처리 방식
### 일괄 처리(Batch Processing)
- 작업을 모아서 한꺼번에 처리한다.
- ex) 초기 Punch Card 처리 시스템

### 시분할(Time Sharing)
- CPU 시간을 분할하여 여러 작업에 나누어 할당한다.
- ex) UNIX

### 실시간(Real-Time OS)
- 정해진 시간 내에 작업 완료가 보장되어야 하는 시스템을 위한 운영체제이다.
- ex) 원자로 제어, 공장 제어, 미사일 제어 시스템
