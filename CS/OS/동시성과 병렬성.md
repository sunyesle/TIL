# 동시성과 병렬성
## 동시성(Concurrency)
- 여러 작업을 **동시에 실행되는 것처럼 보이게 하는 것**
- 단일 CPU 환경에서는 CPU가 여러 작업을 빠르게 번갈아 처리하는 Context Switching을 통해 동시에 실행되는 것처럼 보이게 한다.
- 작업의 실행 순서를 조절하여 CPU 유휴 시간을 줄이고 자원의 효율적으로 활용할 수 있다.
- I/O 작업(파일입출력, API 호출, 데이터베이스 쿼리 등)이 많은 환경에서 효과적이다.

## 병렬성(Parallelism)
- 여러 작업을 **실제로 동시에 실행하는 것**
- 여러 CPU나 코어를 사용하여 여러 작업을 실제로 동시에 실행한다.
- 연산 작업을 여러 부분으로 나누어 처리하기 때문에 처리 시간을 단축할 수 있다.
- 이미지 처리, 대규모 데이터 분석, 머신러닝 학습 등 CPU/GPU 연산량이 큰 작업에 유용하다.
