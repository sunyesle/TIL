## 합집합 - addAll()
`a.addAll(b)`: a에 b의 요소를 모두 추가한다.
```java
Set<Integer> a = new HashSet<>(List.of(1, 2, 3));
Set<Integer> b = new HashSet<>(List.of(3, 4, 5));

a.addAll(b);
System.out.println(a); // [1, 2, 3, 4, 5]
```
## 교집합 - retainAll()
`retainAll()`: a에서 b와 공통된 요소만 남기고 나머지는 제거한다.
```java
Set<Integer> a = new HashSet<>(List.of(1, 2, 3));
Set<Integer> b = new HashSet<>(List.of(3, 4, 5));

a.retainAll(b);
System.out.println(a); // [3]
```
## 차집합 - removeAll()
`removeAll()`: a에서 b에 포함된 요소를 제거한다.
```java
Set<Integer> a = new HashSet<>(List.of(1, 2, 3));
Set<Integer> b = new HashSet<>(List.of(3, 4, 5));

a.removeAll(b);
System.out.println(a); // [1, 2]
```

> ### 주의 사항
> 원본 객체를 변경하므로, 원본을 유지하려면 복사본을 만들어서 연산을 수행해야 한다.
> ```java
> Set<Integer> a = new HashSet<>(List.of(1, 2, 3));
> Set<Integer> b = new HashSet<>(List.of(3, 4, 5));
>
> Set<Integer> aCopy = new HashSet<>(a);
> aCopy.retainAll(b);
>
> System.out.println(a); // [1, 2, 3] (원본 유지)
> System.out.println(aCopy); // [3]
> ```
