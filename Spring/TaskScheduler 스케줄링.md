# TaskScheduler 스케줄링
스프링 부트는 주기적인 작업을 수행할 수 있도록하는 스케줄링 기능을 제공한다.

## application.yml
```yml
spring:
  task:
    scheduling:
      thread-name-prefix: scheduling-
      pool:
        size: 1
      shutdown:
        await-termination: true
        await-termination-period: 30s
```

| 속성                                                        | 기본값         | 설명                                       |
|------------------------------------------------------------|---------------|----------------------------------------------|
| `spring.task.scheduling.pool.size`                         | `1`           | 스케줄러가 사용하는 스레드 풀의 크기            |
| `spring.task.scheduling.shutdown.await-termination`        | `false`       | 애플리케이션 종료시 수행중인 작업 완료 대기 여부 |
| `spring.task.scheduling.shutdown.await-termination-period` | `0`           | 애플리케이션 종료시 수행중인 작업 완료 대기 시간 |
| `spring.task.scheduling.thread-name-prefix`                | `scheduling-` | 스레드 이름 접두사                             |


## 스케줄링 활성화
`@Configuration` 클래스에 `@EnableScheduling` 어노테이션을 추가하여, `@Scheduled` 어노테이션을 활성화한다.
```java
@Configuration
@EnableScheduling
public class SchedulingConfig {
}
```

## @Scheduled
메서드에 `@Scheduled` 어노테이션을 추가하여 스케줄링 될 메서드를 지정할 수 있다.
메서드는 반환 값이 `void`여야 하고, 파라미터를 받아서는 안 된다. 

### FixedDelay
고정된 지연 시간으로 호출된다. (작업이 *끝난* 시점부터 지정된 시간만큼 기다린 후 다음 작업 실행)
```java
@Scheduled(fixedDelay = 5000)
public void doSomething() {
	// ...
}
```
> 다음과 같이 `TimeUnit`을 지정할 수도 있다.<br>
> ```java
> @Scheduled(fixedDelay = 5, timeUnit = TimeUnit.SECONDS)
> ```

### fixedRate
고정된 속도로 호출된다. (작업이 *시작된* 시점부터 지정된 시간만큼 기다린 후 다음 작업 실행)
```java
@Scheduled(fixedRate = 5000)
public void doSomething() {
	// ...
}
```
### fixedRate
메서드의 첫 번째 실행 전에 대기할 시간을 지정한다.

`fixedDelay`, `fixedRate`와 함께 사용할 수 있으며, 일회성 작업의 경우 다음과 같이 초기 지연 시간을 지정할 수 있다.
```java
@Scheduled(initialDelay = 1000)
public void doSomething() {
	// 애플리케이션 시작 후 1초 뒤에 한 번만 실행된다.
}
```

### cron
크론 표현식으로 실행 간격을 지정한다.
```java
@Scheduled(cron="0 0 * * * MON-FRI")
public void doSomething() {
	// 평일 매시간 정각에 실행된다.
}
```

## 주의 사항
기본 스케줄러는 단일 스레드로 실행된다.<br>
**서로 다른 작업**을 **병렬적**으로 실행하기 위해서는 **스레드 풀을 설정**해야 한다.
- **예시 상황**: 작업 A와 작업 B가 존재함
  - **`pool-size: 1` 기본값 사용**: 단일 스레드가 작업 A와 B를 번갈아 가며 처리한다. 작업 A 종료 후 B가 시작된다.
  - **`pool-size: 2`**: 각 스레드에 작업 A와 B를 할당한다. 작업 A와 B가 동시에 시작된다.

<br>

스케줄러는 동일 작업에 대한 순차적 실행을 보장하기 위해 이전 작업이 종료될 때까지 다음 작업이 실행되지 않는다.<br>
**하나의 작업**을 **병렬적**으로 실행하기 위해서는 **`@Async`를 함께 사용**해야 한다.
- **예시 상황**: `fixedRate = 5s`, 작업 시간 `10s`
  - **`@Scheduled` 단독 사용**: 작업이 완료될 때까지 스레드를 점유 중이라 다음 작업이 대기하게 된다. 실제 실행 간격은 10초가 된다.
  - **`@Scheduled`** + **`@Async`**: 작업 실행 즉시 `TaskExecutor`에게 작업을 넘기고 스레드가 반환된다. 비동기 풀에서의 작업 완료 여부와 관계없이 5초 뒤에 다음 작업이 실행된다.

## 크론 표현식
스프링에서 사용되는 크론 표현식은 총 6개의 필드로 구성되어 있다.
```
 ┌────────────── 초 (0-59) 
 │ ┌───────────── 분 (0-59) 
 │ │ ┌───────────── 시 (0-23) 
 │ │ │ ┌───────────── 일 (1-31) 
 │ │ │ │ ┌───────────── 월 (1-12 또는 JAN-DEC) 
 │ │ │ │ │ ┌───────────── 요일 (0-7) 
 │ │ │ │ │ │ (0-7 또는 MON-SUN) 
 │ │ │ │ │ │ 
 * * * * * *
```
### 필드
| 순서 | 필드 | 허용 값                      | 허용 특수문자        |
|----|------|---------------------------|----------------|
| 1  | 초   | 0-59                      | `*` `,` `-` `/`        |
| 2  | 분   | 0-59                      | `*` `,` `-` `/`        |
| 3  | 시   | 0-23                      | `*` `,` `-` `/`        |
| 4  | 일   | 01월 31일                   | `*` `,` `-` `/` `?` `L` `W`  |
| 5  | 월   | 1-12 or JAN-DEC           | `*` `,` `-` `/`        |
| 6  | 요일 | 0-7 (0,7은 일요일) 또는 MON-SUN | `*` `,` `-` `/` `?` `L` `#`  |

### 특수문자
- `*` : 모든 값
- `?` : 특정한 값이 없음
- `-` : 범위
- `,` : 구분자
- `/` : 시작 시간과 간격
- `L` : 월의 마지막 날, 월의 마지막 요일
- `W` : 가장 가까운 평일
- `#` : 몇째주의 무슨 요일

### 예시
| 크론 표현식              | 의미                                       |
|------------------------|------------------------------------------|
| `0 0 * * * *`          | 매일 매시간 정각                                |
| `*/10 * * * * *`       | 10초마다                                    |
| `0 0 8-10 * * *`       | 매일 8시, 9시, 10시                           |
| `0 0 6,19 * * *`       | 매일 오전 6시부터 오후 7시까지                       |
| `0 0/30 8-10 * * *`    | 매일 8:00, 8:30, 9:00, 9:30, 10:00, 10:30  |
| `0 0 9-17 * * MON-FRI` | 평일 오전 9시부터 오후 5시까지 매 시간 정각에               |
| `0 0 0 25 DEC ?`       | 매년 크리스마스 자정                              |
| `0 0 0 L * *`          | 그 달의 마지막 날 자정                            |
| `0 0 0 L-3 * *`        | 해당 월의 마지막에서 세 번째 날 자정                    |
| `0 0 0 * * 5L`         | 매달 마지막 금요일 자정                            |
| `0 0 0 * * THUL`       | 매달 마지막 목요일 자정                            |
| `0 0 0 1W * *`         | 매달 첫째 평일 자정                              |
| `0 0 0 LW * *`         | 해당 월의 마지막 평일 자정                          |
| `0 0 0 ? * 5#2`        | 매달 두 번째 금요일 자정                           |
| `0 0 0 ? * MON#1`      | 매달 첫째 주 월요일 자정                           |
