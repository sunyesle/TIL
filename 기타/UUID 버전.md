# UUID 버전

## 버전 비교
| 버전   | 특징                                 |
|--------|-------------------------------------|
| **v1** | 날짜-시간, MAC 주소                  |
| **v2** | 날짜-시간, MAC 주소, DCE 보안 버전    |
| **v3** | 제공된 데이터의 MD5 해시              |
| **v4** | 랜덤 데이터                          |
| **v5** | 제공된 데이터의 SHA-1 해시            |
| **v6** | 날짜-시간, MAC 주소                  |
| **v7** | Unix 타임스탬프 + 랜덤 데이터   |
| **v8** | 사용자 정의 가능 (버전/변형 필드 제외) |

## 사용 사례
- **v4**: 랜덤 ID가 필요한 경우
- **v7**: 정렬이 필요한 경우 (ex: 데이터베이스 키)
- **v5**, **v8**: 특정 데이터를 UUID에 포함시키고 싶은 경우

## 그 외 정보
- v7은 v1와 v6를 개선한 버전이다. MAC 주소 정보가 없고, 표준적인 타임스탬프를 사용한다.
- v5는 v3를 개선한 버전이다. 더 안전한 해싱 알고리즘을 사용한다.
- v8은 비표준 UUID를 위한 사양으로 버전/변형 필드를 제외한 나머지를 사용자 정의할 수 있다.
